# 🪐 OSW Project (オッスウィー)

> **「中心（まんなか）空っぽ、誰も座らない。三角（ピラミッド）を突き抜け、$Z$の風を起こす分散型アジト」**

---

## 👁️ 理念（Philosophy）

現代の社会システムは、お金と権力を使った洗練された奴隷制度です。
資本主義という三角形（ピラミッド）の頂点に立つ「資本家」、官僚機構という檻の中で行政の長に従う「公務員」。私たちは生存を人質に取られ、本来「他者と生きていくための営み」であったはずの労働を奪われています。

**「俺たちナニエ？ オッスウィー」**

私たちは、誰かに支配される2次元の平面ゲームからフェードアウトします。
目を開き、手のひらを開く。オープンソース（Open Source）という武器を使い、自発的な貢献と共生の循環である**「$Z$軸の未来」**を自分たちの手でビルド（創生）するためのプロジェクト、それが **OSW（Open Source We-use）** です。
