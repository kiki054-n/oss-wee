## 👁️ 理念（Philosophy）

本当の敵は、どこにもいません。

資本家も、公務員も、もともと誰かの敵だったわけではありません。敵は、こちらが「あれは敵だ」と決めつけた瞬間に生まれます。敵を作っているのは、いつも自分自身です。

その「敵を作る力」は、消すべき悪ではありません。生きていくために必要な働きでもある。
「ダメなものはダメ」と止めるとき、人は一度、相手を敵にしなければ押し返せない。
人の中に性善と性悪が同時にあるのは、たぶんこのためです。
同じ一つの力が、自分の利益のために他人へ向ければ攻撃材料になり、ダメを止めるために向ければ、守る働きになる。

だから消すのではなく、区別します。
**「行いを止めること」**と**「人を敵に固定すること」**は、別のことです。
前者は要る。後者を、やめる。
敵だと思った相手も、気づかせることができれば、もう敵ではなくなるのだから。

そして、本当に自分を縛っているものの正体は、外の誰かではありませんでした。
**一人でいること――それが、自分で自分を奴隷にすることだった。** 
友がいなければ、人は生きられない。

「俺たちナニエ？ オッスウィー」

だから私たちは、誰が敵で誰が味方かを奪い合う、2次元の平面ゲームから静かに降ります。
目を開き、手のひらを開く。
平面から、次の次元へ――**Z軸方向へ、みんなで舞い上がろう。この指とまれ。**

オープンソース（Open Source）を手に、
自発的な貢献と共生が循環する**「$Z$軸の未来」**を、
自分たちの手でビルド（創生）する。
それが OSW（Open Source We-use） です。

## 🌌 星を見るように / Why "We", not "I"

なぜ "I Use"（私が使う）ではなく **"We Use"（私たちが使う）** なのか。

ここに共有されているものを、私たちは**宇宙に広がる星を見つめるように**考えなければなりません。

星を見上げたとき、届いている光は何万年も昔のものかもしれない。その星は、もう消えているかもしれない。**ひとりで見上げても、その光が今の真実なのかは、確かめようがありません。**

共有された情報も同じです。答えを手渡されたように見える。でも、その答えが正しいのかは、一人では分からない。

だから――**この情報を、ひとりで使わないでください。**

別々の場所から、別々の角度で、同じ星を見る。「これは正しいのか」を私たちで持ち寄ったとき、ただの光は、はじめて意味になる。ひとりで使った瞬間、それは検証されない孤立した答えに戻ってしまう。

"We Use" は、お願いではありません。共有知が知になるための、構造そのものです。

> 🔭 私たちの星空 → **digital-commons**（共有された実物が置かれる場所）

## 🖐️ この指とまれ / Here. Stop on this finger.

源（source）は、向こうにあるのではありません。
頂点にも、資本にも、許可をくれる誰かにもない。

**いま、あなたの手のひらにあります。**

ピラミッドを壊しに行く必要はありません。
ここに立って息ができたら、あの三角形は、ひとりでにただの影になります。

難しいことは何もいりません。憲章（下）の Hook のとおり、順番に三つ。

### 1. 目を開け（Open your eyes）
この repository を眺める。気になるものを一つ、見つける。

### 2. 汝の手を見よ（Look at your hands）
あなたが今すでに持っているもの――技術、言葉、問い、時間――を一つ、思い出す。

### 3. 指にとまる（Contribute）
下のどれか一つを、実際にやる。

| 動作 | 意味 |
|------|------|
| ⭐ **Star** | 「ここにいるよ」の合図 |
| 🍴 **Fork** | あなたの手の中に持って帰る |
| ✍️ **Issue** | あなたの問いを、ここに置いていく |
| 🔁 **Pull Request** | あなたの一行を、風に乗せる |

どれも「正しい貢献」である必要はありません。
小さく、未完成で、間違っていても構いません。
**握りしめた拳じゃ、何も生めない。手を開いた指が一本増えること自体が、ここでは出来事です。**
